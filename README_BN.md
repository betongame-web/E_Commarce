# অরোরা প্রিমিয়াম স্টোর UI

এটি একটি সম্পূর্ণ responsive Bangla premium e-commerce landing page। মোবাইল ফোন, MacBook, laptop, gold jewellery, diamond jewellery এবং smartwatch-এর জন্য তৈরি।

## চালানোর নিয়ম

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

## প্রধান ফাইল

- `src/App.jsx` — সব section, product data এবং interactive logic
- `src/styles.css` — সম্পূর্ণ premium responsive styling
- `public/assets/` — local product visuals এবং logo

## সুবিধা

- Desktop, tablet ও mobile responsive
- সব UI লেখা বাংলায়
- ৫০% ছাড়ের banner ও badges
- Add-to-cart counter এবং notification
- Wishlist button
- Live countdown timer
- Mobile navigation menu
- Local product assets; external product-image dependency নেই

ছবি ও লেখা নিজের ব্র্যান্ড অনুযায়ী `src/App.jsx` থেকে সহজেই পরিবর্তন করা যাবে।
